## 使用方法

开袋即食，我已经把大部分 markdown 和 tex 内容导出了 pdf，直接在 GitHub 网页浏览即可。一些导航：

- 课程根目录下的 `final.pdf` 是笔记汇总（如果存在）；
- `homework` 或者 `labs` 下包括了所有归档作业（ICS 的小作业在 `symposium` 里）。

如果你需要查看或编辑 markdown 源码（例如一些正在更新的笔记），建议下载相应源码后使用 Typora 查看或编辑，因为我写了很多很多 Typora 方言（？），Github 网页端和 Obsidian 几乎都一定不能正确地渲染。
